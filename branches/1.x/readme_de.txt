Befor sie Skip-Net starten, stellen Sie sicher, dass auf ihren System Java Runtime Environment von Sun installiert wurde, indem sie im DOS-Fenster folgenden Befehl eingeben:
	java -version 
Wenn JRE vorhanden ist und korrekt installiert wurde, dann muss eine kleine Version-Meldung erscheinen, wenn nicht, dann können sie aktuelle JRE von der SUN-Site www.java.com runterladen. 
Starten Sie aus dem WindowsExplorer die skipnet_de.jnlp Datei. Wenn Java korrekt auf Ihrem System installiert wurde dann wird die neueste Skip-Net  Version von Skip-Net -Homepage runtergeladen . Danach wird eine Sicherheitswarnung mit verschiedenen Auswahlmöglichkeiten angezeigt. Sie müssen unbedingt auf "Starten" klicken, damit erlauben Sie dem Programm auf Ihrem Rechner zu laufen. 
Alternativ können Sie die jnlp Datei direkt mit javaws.exe starten. 
Durch die Verwendung von JavaWebstart-Technologie werden Sie immer automatisch die neueste Versionen des Skip-Net  Programmes spielen.
