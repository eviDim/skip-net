Skip-Net Program required Java Runtime 1.4 and above. If you have not installed JRE on your System please download and install the current Java Runtime from the Java Site "http://www.java.com".
To start Skip-Net with "Java Web Start" double klick in WindowsExplorer on the skipnet_en.jnlp file and accept the security warning. 
You should be connected with Internet to run Skip-Net first time.
